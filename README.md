# Weather App

Real-time weather application using OpenWeather REST API.

# Features
- Search weather by city name
- Display temperature, humidity, wind speed, weather description
- Dynamic weather icons from API
- Error handling for invalid cities and API failures
- Responsive design for mobile/desktop

# Tech Stack
HTML5, CSS3, JavaScript ES6, REST API, Async/Await

# Live Demo
https://anumulaakshitha.github.io/weather-app-js/

# Key Learnings
- Implemented async/await for API calls
- JSON parsing and DOM manipulation
- Error handling for 404/401 HTTP responses
- Deployed static site using GitHub Pages

# How to run locally
1. Clone repo
2. Add your OpenWeather API key in script.js
3. Open index.html in browser
